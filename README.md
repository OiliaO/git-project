# پروژه نظرسنجی

## مقدمه
این پروژه یک فرم نظرسنجی ساده است که به زبان جاوا پیاده‌سازی شده است. هدف از این پروژه، ارزیابی تجربه کاربران از نرم‌افزار و جمع‌آوری اطلاعات اولیه از آن‌هاست. این پروژه به صورت تیمی و توسط دو نفر انجام شده است.

## اعضای تیم
- **عضو اول : ایلیا
- **عضو دوم : پارسا

## ویژگی‌های پروژه
این پروژه شامل چهار صفحه اصلی است که هر کدام وظیفه‌ی خاصی را انجام می‌دهند:

1. صفحه اصلی (Main) :
   - نمایش عنوان فرم نظرسنجی.
   - دکمه‌ی شروع برای ورود به صفحه‌ی بعدی.

2. صفحه اطلاعات (InformationPage):
   - جمع‌آوری اطلاعات اولیه کاربر شامل نام، سن، ایمیل و سطح آشنایی با نرم‌افزار.
   - دکمه‌ی "Next Page" برای رفتن به صفحه‌ی سوالات.

3. صفحه سوالات (QuestionsPage) :
   - شامل چهار سوال نظرسنجی با گزینه‌های امتیازدهی از ۱ تا ۵.
   - دکمه‌ی "Next Page" برای رفتن به صفحه‌ی تشکر.

4. صفحه تشکر (ThankYouPage) :
   - نمایش پیام تشکر برای مشارکت کاربر در نظرسنجی.

## تغییرات اعمال شده
در این پروژه، تغییرات زیر اعمال شده است:

1. **تغییر زبان :
   - تمامی متن‌های برنامه به زبان انگلیسی تغییر کرده‌اند تا برای کاربران بین‌المللی قابل فهم باشد.

2. **تغییر فونت :
   - از فونت `Arial` با سایز ۱۶ و استایل `Bold` برای تمامی متن‌ها و دکمه‌ها استفاده شده است.

3. **تغییر تم و چیدمان :
   - رنگ پس‌زمینه‌ی تمامی صفحات به زرد تغییر کرده است.
   - چیدمان کامپوننت‌ها در صفحات به‌گونه‌ای تنظیم شده است که در مرکز صفحه قرار بگیرند و به کناره‌ها نچسبند.
   - از `BorderLayout` و `BoxLayout` برای سازماندهی بهتر کامپوننت‌ها استفاده شده است.
